# Отчет о проделанной работе по модулю Б: Разведочный анализ данных

## 1. Введение

Целью данного отчета является представление результатов разведочного анализа данных с использованием информационной панели. Мы провели исследование продуктовых метрик, создали профили пользователей и проанализировали маркетинговые показатели для определения стратегии компании.

## 2. Построение дашборда

Мы разработали аналитический дашборд с использованием библиотеки Dash, который предоставляет возможность анализировать основные метрики в реальном времени. Дашборд включает в себя следующие разделы:
- Общие метрики продуктовой активности (DAU, WAU, MAU).
- Время, проведенное пользователями в приложении.
- Распределение пользователей по странам, устройствам и рекламным каналам.
- Анализ продаж для онлайн-торговли.

## 3. Исследовательский анализ данных в информационной панели

Мы вычислили и визуализировали следующие характеристики:
- Заинтересованность клиентов в продукте (DAU, WAU, MAU).
- Количество времени, проведенное в приложении.
- Распределение привлеченных пользователей по различным показателям.
- Анализ продаж для онлайн-торговли.

## 4. Создание профилей пользователей

Используя ранее определенную функцию, мы создали набор данных с профилями пользователей и определили следующие характеристики:
- Страны, из которых приходят посетители, и страны, дающие больше всего платящих пользователей.
- Устройства, которыми пользуются пользователи, и устройства, с которых чаще всего заходят платящие пользователи.
- Рекламные каналы, приносящие больше всего платящих пользователей.

## 5. Маркетинговый анализ

Мы рассчитали следующие показатели:
- Общие расходы на рекламу и расходы на рекламу по источникам и по времени.
- Среднюю стоимость привлечения одного покупателя из каждого источника.
- Оценку окупаемости рекламы и ROI с разбивкой по устройствам, странам и рекламным каналам.
- Анализ конверсии и удержания с разбивкой по устройствам, странам и рекламным каналам.

## 6. Выводы и обоснование выбора

- Мы выявили значимые тренды и показатели, которые могут быть использованы для разработки стратегии компании.
- Основываясь на полученных данных, рекомендуется сосредоточить усилия на оптимизации рекламных кампаний в странах, устройствах и каналах, которые приносят наибольшую окупаемость.

## 7. Заключение

Разведочный анализ данных помог представить комплексную картину о продуктовой активности, поведении пользователей и эффективности маркетинговых кампаний. Полученные результаты позволяют сформулировать рекомендации по дальнейшим действиям и оптимизации стратегии компании.
